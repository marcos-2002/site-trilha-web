# Trilha de Aprendizado Web

Aplicação web desenvolvida para organizar e disponibilizar uma trilha de aprendizado em desenvolvimento web, com o objetivo de estruturar e facilitar o acesso aos conteúdos produzidos.

A plataforma centraliza módulos teóricos e aulas em vídeo, permitindo que o usuário acompanhe sua evolução de forma estruturada.


## Conteúdos da trilha

A trilha cobre tópicos fundamentais e avançados de desenvolvimento web, incluindo:

* Introdução ao Desenvolvimento Web
* JavaScript (básico e avançado)
* Front-end com React
* Introdução a servidores web
* Bancos de dados
* Back-end com Node.js e Express

## Tecnologias utilizadas

* React
* JavaScript
* Tailwind CSS
* Vite

## Como executar o projeto

Siga os passos abaixo para rodar o projeto localmente:

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

# Acesse a pasta do projeto
cd seu-repositorio

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```

Após isso, acesse no navegador:

```
http://localhost:5173
```
